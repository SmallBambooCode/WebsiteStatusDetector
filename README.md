# WebsiteStatusDetector
用于在Windows平台批量测试网站（友链）是否能够打开的PySide6程序，兼容系统代理

## 使用以下代码打包：
`pyinstaller --onefile --noconsole --version-file=version.txt --icon=./icon.ico --add-data "icon.png;." client_ui.py`
